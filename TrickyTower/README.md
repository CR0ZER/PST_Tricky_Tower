# TrickyTower
Ce projet a été mené dans le cadre des Projets Scientifique et Technique de l'ESIEA 3A s01.

Nous avons créé un jeu en python similaire a TrickyTower en utilisant les Libraries:
- Twisted pour la partie réseau 
- Pymunk pour la partie simulation physique 
- Pygame et Pygame_menu pour l'afficahge

Pour lancer un partie il faut tout d'abord exécuter le serveur "TrickyTowerServerTemplate.py"
Puis pour jouer de 1 à 4 client.s, lancez "TrickyTower.py"
cliquez sur 'Suivant', attendez vos partenaires puis cliquez sur 'Jouer'

Les contrôles :
'Flèche BAS' ou 'S' -> accélérer la descente
'Flèche GAUCHE ou DROITE' ou 'Q ou D' -> Déplacer sur les côtés
'Flèche HAUT' ou Z -> Faire pivoter de 90°

Le But est d'empiler les pièces afin d'atteindre la ligne d'arrivée



Afin de Configurer l'@IP du serveur distant vous pouvez modifié le champs 'GAME_IP' du fichier "constants.py"
